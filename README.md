# StormBot-Camera

Codebase for the cameras on the University of Arkansas RIOT (Robotics Interdisciplinary Organization of Teams) robot built for the 2025 STORM (Student Tele-Operated Robotics Mission) competition Power Flash. The cameras are used to detect AprilTags for automatic alignment of the robot, and may, in the future, also be able to locate objects based on color.

Used in parallel with Explothusist/StormBot-Codebase and Explothusist/StormBot-Esp32-Codebase. Our robot includes both a VEX v5 Brain and an Esp32 chip.

VEX Codebase: https://github.com/Explothusist/StormBot-Codebase

Esp32 Codebase: https://github.com/Explothusist/StormBot-Esp32-Codebase
